SpaceX Falcon 9 First Stage Landing Prediction — Capstone Project
🚀 Project Overview
The goal of this project is to predict whether the first stage of a SpaceX Falcon 9 rocket will land successfully. Since a successful landing significantly reduces launch costs, predicting the outcome allows for a better estimation of the overall cost of a mission.

🛠 Project Pipeline
Data Collection & Wrangling:

Gathering data via SpaceX API and web scraping (Wikipedia).

Cleaning, handling missing values, and formatting the dataset for analysis.

Exploratory Data Analysis (EDA):

Using SQL to query the dataset and extract key insights.

Visualizing relationships (e.g., Payload Mass vs. Orbit, Success Rate vs. Launch Site) using Pandas, Matplotlib, and Seaborn.

Interactive Visual Analytics:

Building interactive maps with Folium to visualize launch site proximity.

Developing a dashboard using Plotly Dash for real-time data exploration.

Machine Learning:

Building, tuning (via GridSearch), and evaluating classification models:

Logistic Regression

Support Vector Machine (SVM)

Decision Tree

K-Nearest Neighbors (KNN)

Storytelling & Reporting:

Synthesizing all findings into a final presentation to communicate data-driven insights to stakeholders.

🧰 Tech Stack
Language: Python 3.x

Libraries: Pandas, NumPy, Scikit-learn, Folium, Plotly Dash, Beautiful Soup

Database: SQL / SQLite
